## 代码简介

时间格式转换为数组或句子形式。

形如 "hh:mm:ss(.nn)" 的格式转换为形如 "x hours, y minutes, z.nn seconds" 的英文句子。

练习巩固了 Regular Expressions ，Object Oriented Programming，Unit Test 等等。

## 应用场景

应用中使用 ffmpeg 命令得到视频、音频长度后转换为英文/中文描述句。
